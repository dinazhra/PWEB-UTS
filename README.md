##AgroMart Pupuqu
Sistem Informasi Pengelolaan Toko Pupuk.
## AgroMart Pupuqu merupakan sistem yang membantu dalam pengelolaan produk pupuk untuk memudahkan admin dalam melihat data produk.
## Fitur 
- Login (input username & password)
- Dashboard (menampilkan informasi tentang AgroMart)
- Profile (menampilkan informasi data pengguna)
- Pengelolaan (menampilkan daftar produk)
